Triangle Parallelogram

Triangle

  • A = Area of Triangle
  • P = Perimeter of Triangle
  • b = base
  • h = height

    • Area : A = 1/2 x b x h Perimeter : P = b + b + b


Parallelogram

  • A = Area of Parallelogram
  • P = Perimeter of Parallelogram
  • b = base
  • h = height

Area : A = b x h Perimeter : P = 2b + 2c
