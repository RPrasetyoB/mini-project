Website for calculate Area and Perimeter of Triangle & parallelogram

Triangle Parallelogram

Triangle

  • A = Area of Triangle
  • P = Perimeter of Triangle
  • b = base
  • h = height
  • a = side a
  • c = side c

    • Area : A = 1/2 x b x h Perimeter : P = a + b + c




Parallelogram

  • A = Area of Parallelogram
  • b = base
  • h = height

Area : A = b x c Perimeter : P = 2b + 2c
